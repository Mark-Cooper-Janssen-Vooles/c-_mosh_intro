### Arithmetic Operators

add; +; a + b
subtract; -; a - b
multiply; *; a * b
divide; / ; a / b
remainder (modulo); % ; a % b

increment; ++ ; a++ ; aka a = a+1
decrement; -- ; a-- ; aka a = a-1

postfix increment: 
int a = 1;
int b = a++;

when applied, value of a assigned to b. So b is 1. Then increment is applied, so a = 2 and b = 1.

prefix increment: 
int a = 1;
int b = ++a;

when runs, a = 2, b = 2.
